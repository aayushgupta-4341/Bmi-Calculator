# ⚖️ BMI Calculator

A simple and interactive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript.

This application calculates BMI based on user input (height and weight) and displays the corresponding BMI category.

---

## 🚀 Features

- User input for height and weight
- Instant BMI calculation
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- Clean and responsive user interface
- Dynamic result update without page reload

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation & Conditional Logic)

---

## 📐 BMI Formula Used

BMI = weight (kg) / (height (m) × height (m))
